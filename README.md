# IT Executive Portfolio

This repository hosts a professional portfolio website designed for an IT executive.

## Sections

- Profile
- Resume
- Projects
- AI Projects
- Contact

## Features

- Responsive navigation with hamburger menu on mobile
- Smooth scrolling to sections

## Colors

The site uses a dark theme with dark gray, black, and green accent colors.

## Deployment

This project is structured for GitHub Pages. Simply push to the `main` branch and enable GitHub Pages in repository settings.

You can add a `resume.pdf` file at the root if you want the download link to work.

To display your portrait image on the homepage, place your photo in an `images/` folder (e.g. `images/profile.jpg`) and replace the placeholder SVG. The HTML already includes an `<img>` tag with class `profile-photo` that will show your picture.
